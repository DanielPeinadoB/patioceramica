1. Instalar Node.js (Ir a https://nodejs.org/es/download/ descargar la versi√≥n para Windows 10)
2. Instalar NPM (En la consola de comandos o Terminal de Visual Code, ejecutar: npm install npm@latest -g)

REPETIR EN CADA PROYECTO

1.  EJECUTAR NPM        npm init (dar enter y yes)
2.  Instalar SASS       -g sass
3.  Instalar NODE       npm install -D node-sass nodemon
4.  Crear carpeta css y el archivo "estyle.css"
5.  Crear carpeta scss y el archivo " estyle.ccs"
6.  Editar el archivo "package.json" agregando:

"build-css": "node-sass --include-path scss scss/estilos.scss css/estilos.css",
"watch-css": "nodemon -e scss -x \"npm run build-css\""

7.  GUARDAR EL ARCHIVO
8.  Ejecutar            npm run watch-css
PARA CERRAR EL TERMINAL DE SCSS "CTRL+C"
PARA CORRER DE NUEVO    npm run watch-css
